# bookStoreMERN
Minimal crud for storing book data using MERN stack

# To install:
## Clone Repo
```git clone https://github.com/reno99986/bookStoreMERN.git```
## cd into Repository
```cd bookStoreMERN```

# Backend Setup
## cd into backend
```cd backend```
## Install npm package and run the server
```npm i```  
```npm run dev```

# Frontend Setup
## cd into frontend
```cd frontend```

## Install npm package and run the server
```npm i```  
```npm run dev```
